# 视频转广告

一个 Chrome 浏览器扩展，可以在侧边栏中播放当前标签页的视频，并显示实时热搜资讯。

## 主要功能

- **侧边栏播放**：在当前 Chrome 侧边栏中播放活动标签页的视频，同时显示实时热搜列表。
- **画中画窗口**：支持将视频放入独立小窗口播放，可选择不同外观样式。
- **自定义标题**：可设置侧边栏标题。
- **多源适配**：普通 HTML5 视频源可直接播放；加密/流式视频在浏览器限制下可能无法使用。

## 安装方法

1. 下载页面中的 `video2ad.zip`。
2. 解压 `video2ad.zip`。
3. 打开 Chrome 扩展程序管理页面（`chrome://extensions/`）。
4. 开启右上角“开发者模式”。
5. 点击“加载已解压的扩展程序”，选择解压后的 `video2ad` 文件夹。

## 使用方法

1. 在浏览器中打开视频网站并开始播放视频。
2. 点击扩展图标，选择“打开侧边栏”。
3. 在视频播放页点击原视频全屏（不是侧边栏中的视频）。
4. 之后即可随意切换标签页、最小化窗口或调整侧边栏宽度，不影响视频播放。

## 支持平台

适用于大多数使用 HTML5 `<video>` 标签的网站。受 DRM 保护或跨域 iframe 限制的站点可能无法使用。

## 文件说明

- `video2ad/`：扩展程序源代码目录
  - `manifest.json`：扩展配置文件
  - `background.js`：后台服务脚本
  - `content.js`：内容脚本
  - `sidepanel.html` / `sidepanel.js`：侧边栏页面及逻辑
  - `popup.html` / `popup.js`：扩展弹窗
  - `pip-activate.js`：画中画窗口相关逻辑
  - `bridge.js`：页面与扩展通信桥接
- `index.html`：扩展介绍页面
- `video2ad.zip`：已打包的扩展压缩包
- `video2ad.crx`：已打包的扩展程序
